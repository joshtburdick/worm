Expression browser

(Note: this requires a relatively recent browser which supports HTML5.)

The cluster browser provides a way to find genes with expression similar to a gene of interest. It shows a gene's expression, and the expression pattern of other genes, in decreasing order of correlation.


To center the view on a gene's expression pattern, type its name into the text box, and click “Submit” (or press “Enter”.) The page will update, so that the top row is that gene's expression. From here:

  • clicking on a gene's name will recenter the view on that gene

  • clicking on the description field will go to the Wormbase entry for that gene (even if the description is blank)

  • clicking on the cluster number will go a summary of what was enriched in a particular cluster.
